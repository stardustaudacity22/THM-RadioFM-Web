# THM FM – Trouble Honey Music Radio

THM FM is the official web radio page for **Trouble Honey Music FM**, a South African-based online radio station focused on Pinkster and Christian music.

The website is intentionally lightweight, fast, and simple, providing listeners with immediate access to the live radio stream without unnecessary clutter.

🌐 **Live site:** https://fm.troublehoneymusic.co.za/

---

## 📻 About THM FM

- Jou nommer #1 vir Pinkster liedjies
- Live online radio stream
- Clean, centred, mobile-friendly layout
- Not affiliated with Lede in Christus Kerk

The site is built as a **static HTML project** and hosted via **GitHub Pages** with a custom domain.

## 🚀 Technologies Used

- HTML5
- CSS3 (embedded styles)
- GitHub Pages (hosting)
- Zeno.fm (radio streaming backend)

No frameworks, no build tools, no JavaScript libraries — just clean, reliable web tech.

---

## 🔊 Radio Stream

The live audio stream is provided via Zeno.fm:

https://zeno.fm/player/troublehoneymusic


The HTML `<audio>` element is used for maximum browser compatibility.

> Note: Some browsers block autoplay with sound until the user interacts with the page. This is expected behaviour.

## 🛠 How to Run Locally

You can open the site locally without any setup:

1. Clone the repository:

`git clone https://github.com/RAvanTonder/THM-RadioFM-Web.git`

2. Open `index.html` in your browser.

That’s it.

---

## 📌 Hosting & Deployment

The site is deployed automatically via **GitHub Pages**.

- Branch: `main`
- Root directory deployment
- Custom domain configured via `CNAME`

Any commit to `index.html` is reflected live after GitHub Pages finishes deploying.
